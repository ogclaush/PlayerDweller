# Player Dweller

> Become the Cave Dweller and hunt your friends.

Player Dweller is an **unofficial standalone addon** inspired by Better Cave Dweller, allowing players to fully control the Cave Dweller while preserving its appearance and animations.

Unlike traditional addons, **Player Dweller does not require Better Cave Dweller to be installed**. It is a standalone Forge mod built specifically for **Minecraft 1.19.2**.

This project was originally developed for a YouTube video on **Yotoro's** channel.

---

## Features

- Become the Cave Dweller.
- Multiplayer support.
- Original Cave Dweller model.
- Original animations.
- Walking, running, crouching and crawling.
- Emissive glowing eyes.
- Increased health, damage and movement speed.
- Standalone mod (no Better Cave Dweller installation required).

---

## Commands

```text
/dweller
```

Transforms yourself into the Cave Dweller.

```text
/dweller <player>
```

Transforms another player (Permission Level 2 required).

---

## Requirements

- Minecraft Forge **1.19.2**
- GeckoLib **3.1.40**
